# TradeX 🚀

A full-stack stock trading platform inspired by Zerodha, built using React.js, Node.js, Express.js, and MongoDB.

## Live Demo

### Dashboard

### Backend API

https://tradex-u12x.onrender.com

## Features

* Modern Zerodha-inspired UI
* Interactive Trading Dashboard
* Holdings Management
* Orders Management
* Positions Tracking
* Funds Section
* Buy & Sell Stocks
* MongoDB Integration

## Tech Stack

### Frontend

* React.js
* React Router
* Material UI
* Chart.js

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
